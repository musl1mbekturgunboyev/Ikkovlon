⚙️ Sozlash kerak: bu ilova ma'lumotlarni saqlash uchun bepul Firebase Realtime Database'dan foydalanadi. Hoziroq sozlangan emas — quyidagi qadamlarni bajaring:

1. Firebase loyihasini yarating

  1. console.firebase.google.com ga kiring va yangi loyiha yarating (bepul).
  2. Chap menyudan Build → Realtime Database ni oching, Create Database tugmasini bosing, hududni tanlang va test modeda boshlang.
  3. Rules bo'limiga o'ting va quyidagini joylashtirib, Publish qiling:
{
  "rules": {
    "rooms": {
      "$room": { ".read": true, ".write": true }
    }
  }
}

2. Web ilova sozlamalarini oling

  1. Loyiha sozlamalari (⚙️) → Your apps → </> Web belgisini bosib yangi web ilova qo'shing.
  2. Ko'rsatilgan firebaseConfig obyektini nusxalab oling.

3. Fayl ichiga joylashtiring

Shu HTML faylni oching, firebaseConfig deb yozilgan joyni toping (fayl boshida, <script> ichida) va o'zingizning qiymatlaringiz bilan almashtiring, so'ng qayta saqlab Vercel'ga qayta joylashtiring (deploy).

Sozlangandan so'ng bu xabar avtomatik yo'qoladi.
Shu havolani va xona kodini sevgilingizga yuboring — u ham shu kodni kiritib qo'shilsin.
